View -> Status Bar Previous Top
Show or hide status bar.


  • Messages of application
  • Number of selected record
  • Type of file, DOS, MAC or UNIX, in order to know new line characters
    • DOS --> LF + CR
    • UNIX --> LF
    • MAC --> CR + LF
  • Selection mode :
    • Stream selection
    • Column selection
  • Carret position ( line + column)

See View Menu, Text -> Column Selection, Text -> Stream Selection.
