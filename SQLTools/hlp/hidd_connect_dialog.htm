Connect dialog

The dialog allows you to create a new connection or open one from saved connections.

Connection list

You can choose a connection to open from saved connections. Press on the header bar for sorting by any column.

User / Password

Specify the user / password for Oracle account.

TNS

Specifies the TNS alias from TNSNAMES.ORA.

Direct Connection

Allows to format a connection string from Host, TCP Port and SID instead of lookup this information from TNSNAMES.ORA. It can be usefull if you need to look a new instance quickly.

Host / TCP Port / SID

Specify an instance location when Direct Connection box is checked.

Save Password

Allows to store passwords for all saved connection. Password is stored in Windows registry without encryption so it's up to you to use it or not.

Test

Click this button to test connection with the specified connection attributes.

Copy

Click this button to format TNS alias string from Host, TCP Port and SID and copy it to Clipboard. Available only for Direct Connection.

Delete

Click this button to delete the current connection from Connection List.

Ok

Click this button to open a connection and close the dialog box. All connection attributes will be automatically saved.

Cancel

Click this button to open a connection and close the dialog box..

Help

Click this button to show this help.
